# Buku Tamu Digital

Aplikasi buku tamu untuk Stasiun Geofisika Nganjuk.

## Cara Menjalankan

### Versi Streamlit
```bash
pip install streamlit pandas
streamlit run buku_tamu.py
```

### Versi PHP
- Upload ke hosting yang mendukung PHP
- Pastikan folder `photos` memiliki izin tulis
